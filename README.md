## üìå Project Overview
This project focuses on analyzing car sales performance using an interactive Power BI dashboard.  
The dashboard helps dealerships understand sales trends, pricing behavior, regional performance, and product demand to support data-driven decision-making.

---

## üõ†Ô∏è Steps Followed

### 1. Data Collection & Import
- Imported car sales data into Power BI
- Verified data accuracy and consistency

### 2. Data Cleaning & Preparation
- Removed duplicate records
- Handled missing and null values
- Standardized date formats and categorical fields
- Created a Calendar Table for time-based analysis

### 3. Data Modeling & Transformation
- Built relationships between Calendar table and Car Sales data
- Used Power Query for ETL and data transformations

### 4. Data Analysis & DAX Calculations
- Created key measures using DAX:
  - YTD, MTD, YOY metrics
  - Growth percentage calculations
  - Previous Year comparisons

### 5. Dashboard Development (Power BI)
- Designed an interactive and user-friendly dashboard
- Implemented slicers for dynamic filtering
- Created Overview and Details report pages

## üìä Dashboard Pages

### üîπ Overview Page

### üîπ Details Page

### Sales Performance

- MTD Sales: $54M+
- YOY Sales Growth: ~23%

### Average Price Analysis
- Average Car Price: $28K
- YOY Price Change: Slight decline (~1%), indicating competitive pricing

### Cars Sold
- Total Cars Sold (YTD): 13K+
- YOY Growth in Cars Sold: ~19%

- Certain dealer regions consistently outperform others
